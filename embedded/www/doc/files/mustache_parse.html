Description

Welcome to the Mustache project, written by Andreas Kupries.

For availability please read Mustache - How To Get The Sources.

This package provides the parser for mustache templates.

Commands

::mustache parse template ?delimiters?

Parses the mustache template, and returns a Tcl data structure representing it.

The details of the data structure are not documented by design, it is not relevant to users of the command.

All it is necessary to know is that the command provided by package mustache::render takes such a structure and renders it, using a context as the source of the data to interpolate.

By default the parser will use the strings {{ and }} to detect beginning and end of mustache tags. This can be overridden using the delimiters argument, by providing a 2-element list of tag opener and closer strings, in this order.

Note however that this can also be done by prefixing the template string with the proper mustache tag for changing the delimiters.
